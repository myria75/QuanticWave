Patterns:


- Initialization: Since only Pauli gates are involved, by checking the beggining of each qubit and look for Pauli X, Y or Z gates the pattern can be detected.

- Superposition: Only Hadamard gates are needed in all qubits and check that each qubit superposition has not being cancelled previously.

- Uncompute: Aparte from being symmetric the qubits must end in the same state than before applying the circuit. To detect this, a simulation or execution is needed to find out this condition.

- Entanglement: A sub-get of qubits should be superposed. In later columns, CNOT gates should be applied.
